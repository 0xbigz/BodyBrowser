BodyBrowser
===========

####Start Local Server

Python: python -m SimpleHTTPServer

PHP: php -S localhost:8080

#####or Enable CORS

Chrome: "open /Applications/Google\ Chrome.app --args --allow-file-access-from-files"

Safari: Menu > Develop > Disable Local File Restrictions
